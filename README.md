# RC-plane-frame
This project documents the design and construction of a fixed-wing RC aircraft platform. The focus was on structural design, component integration, and manual flight testing.

## System Description
- Manual RC control (no onboard flight controller)
- Standard servo-actuated control surfaces
- Brushless motor + ESC propulsion system

## Components
- RC transmitter and receiver
- Servos (control surfaces)
- ESC + brushless motor
- Battery system
